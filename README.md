# React Recycler View (⍺)

Common library for lists virtualization. Provides basic set of tools to setup any custom layout with virtualization.

Layouts included:

- Vertical lists
- Masonry (in progress)

Modes supported:

- Normal container
- Container with `overflow: scroll`
